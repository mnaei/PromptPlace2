Welcome to the LLM Editable Webpage

This webpage can be modified using natural language instructions. Simply enter your request in the form above, and our system will use the Gemini API to automatically update this page based on your instructions.

How It Works

  1. Enter your instructions in the text box above
  2. Click "Submit" to create a GitHub issue with your request
  3. A GitHub Actions workflow will process your request
  4. The workflow uses Gemini to convert your instructions into HTML/CSS/JS
  5. The changes are automatically committed to the repository
  6. This page will be updated to reflect your requested changes

Example Instructions

  • "Add a photo gallery section with 3 placeholder images"
  • "Change the background color to light blue and add a footer"
  • "Create a navigation menu at the top with Home, About, and Contact links"
  • "Add a contact form with name, email, and message fields"

Note: Before using this page in production, remember to update the form action URL with your actual GitHub repository details (replace REPO_OWNER/REPO_NAME).
